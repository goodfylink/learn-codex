# Rust Review Skill

Use this skill when the user explicitly asks for a Rust code review with `$rust-review`.

Workflow:
- Start by reading the files that are most likely involved in the request.
- Focus on correctness, regressions, edge cases, and missing verification.
- Prefer concrete findings over general praise.
- If no issue is found, say that explicitly and mention any remaining testing risk.
- Keep the final answer concise and practical.
